Campaign Data Visualization

Project completed: April 2015

Tools used: HTML/CSS, Twitter Bootstrap, JavaScript/jQuery, AJAX, HighCharts

This project was completed this project as a part of a class assignment for JOMC 586 (Intermediate Interactive Multimedia). The project uses data from the Sunlight Foundation and OpenSecrets to compare the contributions received by N.C. congresspeople in the U.S. House of Representatives in 2013.
